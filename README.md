# DuoPython: Интерактивный тренажер языков

**Автор:** [tipoCrutoi228(мой ник на ХАБР)]

DuoPython — это десктопное приложение для изучения иностранных слов, вдохновленное механиками Duolingo. Проект создан на Python с использованием графической библиотеки Tkinter и технологий распознавания речи.

---

## 🎯 Описание проекта
Цель проекта — сделать процесс изучения слов более динамичным и вовлекающим. В отличие от обычных словарей, DuoPython заставляет пользователя не только выбирать правильные варианты визуально, но и практиковать произношение, что критически важно для запоминания языка.

## ✨ Функции и особенности

### 🌍 Игровая карта
* **5 уникальных локаций**, каждая из которых открывается по мере прохождения.
* Всего **25 уровней**, предлагающих постепенное усложнение материала.

### 🎮 Игровые механики (5 уровней на локацию):
* **Текстовые уровни (1-3):** Классический тест. Вам дается слово, и нужно выбрать один из двух вариантов ответа.
* **Голосовые уровни (4-5):** Самая сложная часть. Программа выводит слово, а вы должны произнести его перевод в микрофон. Система сравнивает вашу речь с правильным ответом.

### 👤 Личный профиль:
* **Система аккаунтов:** Регистрация и авторизация пользователей.
* **Сохранение прогресса:** Ваши очки и открытые локации привязаны к вашему профилю.
* **Очки опыта:** За каждый правильный ответ начисляются баллы.

## 🛠 Установка и использование

### Требования
Для работы голосовых уровней необходим Python 3.x и библиотека для работы с микрофоном.

### Инструкция
1.Установите нужные библиотеки это :
  '''pip install speechrecognition
  pip install translator
  pip install tkinter'''
2. запустите :
  '''python duolingo.py'''
